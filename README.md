# 葉冠志(peter)的暑假日誌

我是葉冠志，英文名字是Peter，國小剛畢業，進入錦和高中國中部就讀。


## 2023-06-23 (五)
今天我打算去新北市立圖書館總館看書，所以我跟我爸借youbike騎腳踏車。一開始還沒事，但騎到一半就開始下大雨，因為我沒穿雨衣，所以全身都濕透了。那時我們在水果店，所以我們買了一堆快爛掉的香蕉and荔枝，我們一邊吃4根香蕉，一邊等雨小一點，我們想說既然都濕了，那就衝吧！但我們錯了，雨越下越大，我們只好在一家麵線店門口停下，再吃一碗熱騰騰的綜合麵線再回家。麵線店的老闆娘很貼心的送我一件雨衣，好讓我們回家。我們在小巷子裡穿梭，經過了淹水的路時，腳直接泡在水裡，我爸還倒了三次水呢! 突然在一個離家100公尺的轉彎，我的車子打滑，我載的荔枝飛了出去，我爸趕緊跳車，查看我的傷勢，前面的香蕉也摔爛了。(托雨衣的福，我沒受傷) 然後我就先走回家，我爸就去還車，就這樣結束了一天～

## 2023-06-24 (六)
今天我再次打算去新北市圖書館總館，但老天好像不太想要我們出門，才剛穿好襪子，一踏出門檻，就打雷了。我媽笑道:「老天真的不讓你們去圖書館呢！」但這場雨讓我想到還有金庸小說可以看呢！順便一提握現在看的是[金庸小說](https://zh.wikipedia.org/zh-tw/%E9%87%91%E5%BA%B8)的[飛狐外傳](https://zh.wikipedia.org/zh-tw/%E9%A3%9E%E7%8B%90%E5%A4%96%E4%BC%A0)，推薦大家看飛狐外傳前可先看[雪山飛狐](https://zh.wikipedia.org/zh-tw/%E9%9B%AA%E5%B1%B1%E9%A3%9E%E7%8B%90)喔！扯回來！我翻開第不知道多少頁時，突然想起當年(一年前)第一次碰金庸的原因。事情是這樣的：五年級的某個星期一，我們班導那時有事，所以就請了代課老師，那一天剛好有閱讀課(去學校圖書館的課)，代課老師就帶我們去圖書館。進去時，我就想說有甚麼書比較好看，於是我就問代課老師有啥推薦的，老師想了一下，就指了指旁邊的金庸說，我推薦這一套。我拿的第一本是[神鵰俠侶](https://zh.wikipedia.org/zh-tw/%E7%A5%9E%E9%B5%B0%E4%BF%A0%E4%BE%B6)一翻開第一頁就立刻沉浸在小說裡，急忙道謝就開始看了！好！既然都介紹完了，我們就下次再見囉～

## 2023-06-25 (日)
今天我跟我爸一起去錦和運動中心，詢問桌球家教的費用和可以的時間，那裏的人說會再聯絡，然後我們原本打算要直接回家。但我跟我爸說，Youbike現在只有騎不到5分鐘，所以我們再次前往[圖書館總館](https://www.library.ntpc.gov.tw/)，放心啦！這次沒下雨。但我們騎著騎著就迷路了，我們竟然花了快一個小時才到！一進去裡面，淡淡的書香味撲鼻而來。裡面真的超大，有一堆有趣的書！它總共有八樓，我看了看，找到了漫畫區的樓層，它在四樓。我趕緊坐電梯到四摟，左找又找終於找到了，它在一個角落。漫畫區有一個自己的房間，裡面全是漫畫，我爸跟我說只能看十五分鐘(因為他忘了帶他的電腦來)，我就找了找，隨便看了本就沉浸在書中。15 minute later～ 我爸很準時的進來，叫我跟他回家，雖然很依依不捨，但也只能等下一次了。

## 2023-06-26 (一)
今天我去補習班，因為昨天我爸去問補習班啥時上課，所以得知從今以後，星期一到五，一點到四點四十都要補習！所以差不多就沒暑假了！讓我來講講今天在發生了啥事～今天我爸就給我一百多塊去買午餐，我去Seven買了一碗涼麵，趕緊吃一吃就趕去補習班。結果一進去發現，提早了十五分鐘，我就刷指紋上樓，對了忘了說這個補習班。這個補習班它很特別，是個團體班制，進去出去都要刷指紋，好讓家長知道小孩啥時進出補習班。一上樓，我看了看我要做的位置，好好坐好。首先呢，補習班的總導師先介紹一下自己，他名叫自強，綽號為大帥。聽說因為他自我介紹時，想寫他是大師，但少寫一橫，所以叫做大帥了。他說了一句話讓我印象深刻，那就是「我就是王道，我說的話就是真理!」大家笑了一下，就要開始上課了。這一天教的是數學，由楊逸的學子宥威老師。他講得十分有趣，今天最大的收穫應該是背 20~100 的質數口訣:

## 2023-06-27 (二)
今天我補習班請假，去銘傳大學，我爸是個大學教授，所以當然可以進去啊！今天我跟我爸進去他的辦公室後，他就開始用電腦，我就躺在軟骨頭上面看金庸的笑傲江湖，我看到一半時，我爸就突然說現在有三種選擇。第一種是先睡覺，起來再打桌球；第二種是去環北橫，但很晚才會到家；第三種是去看電影，再回來打桌球。而我選擇第三種。我們今天去看了明天就要下檔的，蜘蛛人：穿越新宇宙。我推薦家裡有青春期的小孩的家庭一起看。(以免把劇情說出來，所以只好不說了)回到學校後，我爸叫我跟老師們打桌球，他自己要先睡一下。去到體育館，我看到兩位老師，但我不敢進去，直到我被發現為止。(在那之前又進去一個了)(我不知道他們叫甚麼名字)後來跟他們打了一下，發現他們都好厲害，讓我好生佩服。

## 2023-06-28 (三)
今天爸爸到學校，沒有在家裡，還好有金庸小說可以陪我。聽爸爸說，今天在學校遇到大葉老師、徐武孝老師、黃炳森老師，他們說他們以前在唸五專、國中、高中時，也都有看過金庸小說。他們說現在的小朋友可以直接在圖書館借到金庸小說，但是以前的圖書館都沒有，大家都搶著看。他們看的第一本也是「神鵰俠侶」，真的是太巧了。

今天下午爸爸還去中和國民運動中心，幫我問「桌球教練」的事，承辦人今天有連絡到教練，想約下週四早上11點第一次上課。可是當天爸爸媽媽都要去學校，我要自己帶9000元去繳報名費，自己去上課，可憐哪。後來爸爸請運動中心幫忙連絡教練，晚上8點多連絡到教練，說明天早上11點就可以開始第一次上課了。希望明天的桌球課程可以順利學到很多。

## 2023-06-29 (四)
今天早上十一點我去運動中心去上桌球課，我爸先帶我去運動中心等教練來。我爸原本不是說要叫我一個人去嗎？是這樣的，我爸今天早上說，因為他不知道要怎麼聯絡教練，所就陪我一天。我看到教練第一眼時，沒什麼感覺。這天我帶的是顆粒拍，但教練說一開始先用普通的拍子就好，於是他就借我他的。教練一開始先跟我打一球，知道我的實力後，開始餵球，先練我的反拍。我的反拍還算不錯，畢竟我平常都用反拍打嘛！接下來換正手，我的正手打的爛到爆，連幼稚園的都可以贏我，但教練一矯正我打的就變得行雲流水。教練說我有一個壞習慣，就是手會勾、會滑，不是用推的， **所以自然會不自然。最後教練就叫我一正一反打，打得超自然的** ，也不用太大力。一開始我從一個正手超爛，變得行雲流水；從打很急、很大力，到一個使力剛好的過程。

## 2023-06-30 (五)
今天我跟我爸在家裡等黑貓宅急配送包裹去銘傳，沒去補習班。一到了銘傳，我跟我爸就先去吃飯，再去他的辦公室。我躺在軟骨頭上看金庸，我爸一進門就在用電腦。我看了一、兩個鐘頭後，就跑去跟老師們打桌球。外面下著雨，我跟我爸就隨便「借」了把傘去桌球室。我們到了的時候裡面沒人，所以我爸就先餵球給我練習，順便等人來。五分鐘後就慢慢的有人來了，我一開始先跟大葉老師打十一分，他讓我八分，然後彥輝哥個打賭，如果我贏大業老師，他就給我他的很貴的新手拍，雖然只有黏一面的皮，但拍子好像很貴，所以我才險勝，自然就拿到了啊。第二次我跟陳主任打，她讓我六分，我打得還算輕鬆，畢竟前面已經把手感找回了，後來她不信邪不信邪又跟我打了一局，她還是輸，她最後不讓分，還是輸，呵呵。後來我又跟其他老師打，其他的都有讓分，但我都贏，好像有點自大，哈哈！果然有去找教練有差，雖然目前只有去一天，但我感到進步的滋味，哈哈！

## 2023-07-01 (六)
今天爸爸媽媽出去，留我跟我哥去補習班。(我今天要補昨天的課)他早在三十分鐘前到補習班，因為他不是補課，而是真的要上課，於是我也匆匆忙忙的出門了。一進到補習班，我跟老師們說我要補課，老師們遲疑了一下，就叫我稍後。我今天上的是社會課，老師講話還蠻有趣的，雖然社會不算是我的強項，但聽他講，以前沒搞懂的，似乎懂了。我上完課以後，我原本想要先回家跟堂哥玩，但老師說，「你媽叫你留在這等你哥」，那時十二點，我聽到時，宛如晴天霹靂般的想說，我哥五點十分才下課！！於是老師叫我回家拿書來看，我回家拿了笑傲江湖的七、八來看，也就是笑傲江湖的最後兩集，我看著看著，兩本都看完了！！後來閒閒沒事做，就開始發呆。五分鐘後，我哥終於下課了，他繳一下學費，我跟他就一起跑回家，就過完了耍廢的一天～

## 2023-07-02 (日)
今天我哥突然說他也想要去大同圖書館，所以我就跟他出發了。走到一半，我就叫我哥等我一下，我要去逛小北百貨。我進去以後，我逛了一下，看到有斜背包，而寫只需要99元！所以我就買了。到了圖書館，我哥就坐下來寫自修，我在旁邊看帶來的俠客行，我看完了以後，我就找一找，找到了我所有金庸唯一還沒看的鹿鼎記。我看了看，想說回家再看好了，所以我就問我哥說要回家了嗎？我哥回答我說，好呀。回家的路上我們遇到了美廉社，我們就進去逛一逛，然後就買了十二個布丁。到家後我們全家就各吃了一個後，打算剩下的以後再吃，就過完了和諧的一天。

## 2023-07-03 (一)
今天我哥早上幫我跟我爸炒炒飯，用昨天沒吃完的純天然的「蕃茄肉醬」和「飯」，炒出了一鍋「黑暗料理」，吃完了以後，我跟我哥就一起去補習班上課。我看了看時鐘，好像早了十五分鐘，所以我就聽老師廢話一下，就開始上國文課，國文老師上課時有趣，使得大家笑得肚子痛，如果我沒記錯的話老師好像叫羽凡老師，她看起來很文青、嚴肅，其實她很搞笑，但該嚴肅的時候，也是會比較兇一點。上完課，我就跑回來家裡，發現昨天的GitHub沒寫，所以昨天的事今天寫的喔！呵呵！

## 2023-07-04 (二)
今天我跟我哥去補習班之前，我們無所事事地做自己的事，結果莫名其妙的，就已經一點十五了。我就衝衝忙忙的出門，我個因為還在整理東西，所以比較晚一點。我出門後，我去全家買了一個三角飯糰和一個原味奶茶吃。吃完後，我趕緊跑去補習班，因為快遲到了。我進去補習班刷完指紋後，就上樓了。今天上的課是英文課，一進去教室後，剛好要考試，考英文。英文是我的強項，所以我考了九十五分，是全班最高分，考完試以後，就開始上課了。我們的英文老師畫畫技術十分「優秀」，老師上課也很有趣，雖然我全部都會了，但不得不說，這老師講話很像ＣＤ裡念的人，上完課我就回家了。

## 2023-07-05 (三)
今天起床的時候已經下午，我想說，我爸不是說要帶我去桃園嗎？但我爸不知道為啥先去了，所以我趕快吃完早午餐，去補習班。到了補習班，刷完指紋，就進去了。今天沒有上課，整天都要考試，所以就考了英文和數學。但我忘了帶金庸，所以在個人指導的時候就很無聊。我忘了說，我數學考九六，英文考九五，算高的。我錯的都是粗心，數學我為什麼錯？因為我錯的是語文問題，例如，超過四是什麼意思，就是從五開始，但我從四開始數，所以就九六了。英文更不應該，我就不說了。好吧，就說到這裡，我們明天見，掰掰。

## 2023-07-06 (四)
今天我起床的時候已經十點了，我起來吃一個麵包，一個車輪餅，看一看金庸的鹿鼎記，然後就要去找桌球教練打桌球。我一進運動中心已經十一點整了，時間過得還真快呢。我趕快進桌球室，看到教練在旁邊處理其他事情等我。教練一看到我，就叫我把東西放下，拍子拿給他看，他問了我一些事情後，借了我一隻球拍，然後開始複習上禮拜的動作。後來他教我腳步，又叫我發球給他看，他說我正手發球有問題，矯正一下我的姿勢。他示範了一下發球，發得超快的，我根本接不到。後來慢慢的我接到了，還打了回去，接下來就開始對打，我幾乎打不贏，原來當教練的水準著麼好，打到十二點，就回家準備去補習班。教練今天說我打得比上次好，果然有練有差，皇天不負苦心人，好期待下次去桃園跟老師們打桌球呢，呵呵。

## 2023-07-07 (五)
今天早上我爸突然把我叫醒，說要帶我去黏桌球拍的皮(請看六月三十號的日記)，我就興高采烈的跟我爸出門，我們騎著腳踏車出門來得了一個台灣桌球代理商的地點，因為是林教練推薦的。一進到裡面就看到了他們在搬桌球桌，我們在裡面跟那裡的人說，我們是林教練跟我們說這裡可以黏桌球皮才來的，他問我們要哪一種皮，我們就說六百塊的那種，他們就去準備，叫我們在一個房間等，在那個房間裡，我們推測他們是有錢人，有著一堆奇怪的東西，還養了一隻狗和一隻烏龜，我們等了一下終於好了！於是我們付完錢就要走了，我們走到樓梯間，那個人問我們要不要球套，我們就上去拿了一個球套回來，聽我爸說，他們竟然沒有收服務費，真的是一個好心的店家。
